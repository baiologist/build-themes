The use of grid layouts in blogs are especially popular on the Tumblr platform, notably on Tumblr’s many photo blogs. For the sample theme, a jQuery library called Salvattore is used create this popular layout.

Adding plugins

The Salvattore library is used to remove the gaps which occur when floating in CSS to create a dynamic grid layout.

JavaScript libraries used in a theme can be uploaded to as a theme asset, and put before the closing body tag of your HTML document:

<script src="//static.tumblr.com/xeccbfq/Qgqnv9yqb/jquery.min.js"></script>
<script src="//static.tumblr.com/xeccbfq/V7Fnvbb38/libs.min.js"></script>

“libs.min.js” includes the multiple JavaScript libraries used in this book. They have been minified and combined into a single file to aid performance, by reducing the number of HTTP requests.

Using Salvattore requires updating both the HTML and CSS of the theme.

Updating the HTML

Update the HTML to signal to Salvattore which elements should be used as the columns and grid container.

{block:IndexPage} and {block:PermalinkPage} are used so only the grid layout is shown on the index, and a column layout is shown on the permalink page:

<main role="main" {block:IndexPage}data-columns{/block:IndexPage}>
    {block:Posts}
    <article {block:PermalinkPage}class='js-column'{/block:PermalinkPage}>

Updating the CSS

Salvattore is largely CSS driven and the column widths can be flexibly set so the theme remains responsive:

  main[data-columns]::before {
	content: '4 .js-column.js-size-1of4';
  }
  main {
	margin-left: 1.75%;
	margin-right: 1.75%;
  }
  .js-column {
	float: left;
	margin-left: 0.75%;
	margin-right: 0.75%;
  }
  .js-size-1of1 {
	width: 98.5%;
  }
  .js-size-1of2 {
	 width: 48.5%;
   }
  .js-size-1of3 {
	width: 31.833333%;
  }
  .js-size-1of4 {
	width: 23.5%;
  }
  @media screen and (max-width: 1200px) {
	main[data-columns]::before {
	  content: '3 .js-column.js-size-1of3';
	}
  }
  @media screen and (max-width: 900px) {
	main[data-columns]::before {
	  content: '2 .js-column.js-size-1of2';
	}
	.meta-wrapper, .nav-wrapper {
	  display: inline-block;
	}
  }
  @media screen and (max-width: 550px){
	main[data-columns]::before {
		content: '1 .js-column.js-size-1of1';
	}
  }

Defining the CSS for the grid layout was largely a matter of following the documentation available on the Salvattore website. When setting the widths of the columns, the margin’s applied to the main and .js-column, should be subtracted to allow for a balanced grid look.

The names of main[data-columns] and .js-column are from the defined the HTML above.
