One of the positives about Tumblr themes is the ability to quickly change your theme. From the customize panel, users can change colors, add images, custom text, add/remove parts of the theme and even change CSS properties.

Customization is available in the theme documentation under appearance options.

All customization options are contained within a HTML meta element with defaults passed through the content attribute.

Colors

Colors can be changed with a color picker from the customize panel by referencing the meta element in HTML and the name in the CSS.

Possible use: letting the users change the theme’s background color.

<!DOCTYPE html>
<html>
<head>
<meta name="color:Background" content="#f4fbfa" />
<style type="text/css">
body {
 background: {color:Background};
}
</style>

Fonts

Tumblr provides a set of web-safe fonts that users can select from.

Possible use: give users the option to select their favorite font.

<!DOCTYPE html>
<html>
<head>
 <meta name="color:Background" content="#f4fbfa" />
<meta name="font:Body" content="Arial, Helvetica, sans-serif"/>
<style type="text/css">
body {
 background: {color:Background};
 font-family: {font:Body};
}
 </style>

Images

Images can be uploaded to Tumblr by users to include their own images for use as a background, masthead or decorative image.

When passing a default value, upload your image as a theme asset in the customize panel.

Possible use: setting the background image for the theme.

<!DOCTYPE html>
<html>
<head>
<meta name="color:Background" content="#f4fbfa" />
<meta name="font:Body" content="Arial, Helvetica, sans-serif"/>
<meta name="image:Background" content="http://static.tumblr.com/wgijwsy/O9sm6ds8v/subtle_zebra_3d.png"/>
<style type="text/css">
body {
 background: {color:Background} url('{image:Background}') repeat;
 font-family: {font:Body};
}
</style>

Conditional Blocks

Developers can create their own conditional blocks which load on the condition that the user selects the option.

In the following example, the welcome text is only rendered when the value is set to 1. To turn it off, by default, set it to 0.

Possible use: displaying a welcome message, if the user wants to.

<!DOCTYPE html>
<html>
<head>
<meta name="color:Background" content="#f4fbfa" />
<meta name="font:Body" content="Arial, Helvetica, sans-serif"/>
<meta name="image:Background" content="http://static.tumblr.com/wgijwsy/O9sm6ds8v/subtle_zebra_3d.png"/>
<meta name="if:Show welcome text" content="1"/>
<style type="text/css">
body {
 background: {color:Background} url('{image:Background}') repeat;
 font-family: {font:Body};
}
</style>
</head>
<body>
{block:IfShowWelcomeText}
<p>Welcome to my blog</p>
{/block:IfShowWelcomeText}

Custom text

Custom text comes in handy when dealing with user-specific text (such as an ID used in a Google Analytics script, or for adding custom text and titles masthead).

Possible use: letting the user write their own welcome text (continued from previous possible use).

<!DOCTYPE html>
<html>
<head>
<meta name="color:Background" content="#f4fbfa" />
<meta name="font:Body" content="Arial, Helvetica, sans-serif"/>
<meta name="image:Background" content="http://static.tumblr.com/wgijwsy/O9sm6ds8v/subtle_zebra_3d.png"/>
<meta name="if:Show welcome text" content="1"/>
<meta name="text:Welcome text" content="Hello world!"/>
<style type="text/css">
body {
 background: {color:Background} url('{image:Background}') repeat;
 font-family: {font:Body};
}
</style>
</head>
<body>
{block:IfShowWelcomeText}
<p>{text:Welcome text}</p>
{/block:IfShowWelcomeText}

Custom CSS

{CustomCSS} is where users can add their own CSS without editing the HTML file. It is hidden under the advanced settings of the customize panel.

If you plan on submitting to your theme to Tumblr, be sure to include the Custom CSS variable.

Possible use: let users customize their theme with CSS!

<!DOCTYPE html>
<html>
<head>
<meta name="color:Background" content="#f4fbfa" />
<meta name="font:Body" content="Arial, Helvetica, sans-serif"/>
<meta name="image:Background" content="http://static.tumblr.com/wgijwsy/O9sm6ds8v/subtle_zebra_3d.png"/>
<style type="text/css">
body {
 background: {color:Background} url('{image:Background}') repeat;
 font-family: {font:Body};
}

{CustomCSS}
</style>
