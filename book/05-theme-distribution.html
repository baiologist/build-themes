Tumblr Themes is the official theme distribution site for developers and users.

The system allows for an easy one-click installation of themes, and a centralized system for updating themes, where all users of your theme automatically receive updates if they have not edited the HTML of the theme.

Submitting themes

Submitting a theme is a two-step process where you submit the theme, then wait for approval or feedback from Tumblr.

Before submitting, make sure your theme meets the guidelines provided by Tumblr. Additionally, have the following information ready to add to the theme submission form:

  • theme title,
  • theme tagline,
  • screenshots (630 x 450px exactly),
  • thumbnail (375 x 250px exactly),
  • demo blog URL,
  • theme support email, and
  • source code or link to a public Github repo with the source code.

Managing Themes

The theme management website, for after you have submitted your theme, allows you to:

  1. View recent active installs
  2. Access statistics about theme installation
  3. Edit or delete a submitted theme
