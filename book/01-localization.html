When developing a blog theme, you will often display text in the user interface, such as “next page” in pagination and “posted on” in a blog post’s metadata.

Since Tumblr can be used in multiple languages including French, Japanese, Portuguese and Spanish, it is important for the theme to display text in the relevant language to the user.

Tumblr solves this problem with localized text, which are a set of commonly-used words which have been translated by Tumblr. Tumblr says “ideally, no language-specific writing should appear in Tumblr themes”.

Localizing follows a familiar pattern of {lang:x}, replacing x with the word you are translating in the HTML markup. Only words which appear in the list of localized words can be used.

Here is an example with no localization on the first line and localization on the second line:

<h2>About me</h2>
<h2>{lang:About me}</h2>

The second line of HTML is rendered in the language the blogger has set, while the first line of HTML always displays “About me” in English.
