Tumblr is a blogging platform, owned by Yahoo, which lets bloggers share anything. Tumblr has experienced phenomenal growth since 2007, with over 250 million blogs and over 100 billion posts online. The platform is used by everyone, from large companies to creatives, to create and share.

Tumblr themes are reusable custom HTML documents used by Tumblr blogs to change the way blogs look and perform on the web. Just like blogging on Tumblr, it is easy to quickly start creating themes.

Creating themes with Tumblr requires prerequisite knowledge of HTML and CSS, so it is important that you are up to scratch with these languages.

This book assumes prior knowledge of HTML and CSS. There is also section which makes use of JavaScript to increase the interactivity of themes. However, JavaScript is not absolutely necessary to create Tumblr themes.

Keep Tumblr’s theme documentation handy as you work through this book, as there are references to the documentation throughout. The theme documentation provides a complete reference of Tumblr theme development.

Editing themes

To make changes to Tumblr theme code, follow these steps:

  1. Click the user settings icon
  2. Select your blog from the blog list
  3. Click edit appearance
  4. Click edit theme

This opens the customize panel. Once you’re in this panel, click on edit html to open up the Tumblr code editor.

The editor provided by Tumblr isn’t great for editing code, so many developers simply copy and paste their code into Tumblr from a text editor, such as Atom.
